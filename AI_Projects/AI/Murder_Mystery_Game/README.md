# Murder Mystery Game

Solve a murder in a Victorian mansion. Explore rooms, find clues, interrogate suspects, and make your accusation before running out of actions. The killer is randomized each game. Suspects are interviewed using AI.

## Setup

```bash
npm install
cp .env.example .env
# Add your Groq API key to .env
npm run dev
```

Get a free API key at [console.groq.com/keys](https://console.groq.com/keys)

Then open [http://localhost:3000](http://localhost:3000)
