# BMO Console Buddy

A fun, interactive console buddy inspired by BMO from Adventure Time. Play games, toggle screens, and listen to lofi music.

**Disclaimer:** This is a fan-made hobby project. Not affiliated with or endorsed by Adventure Time, Cartoon Network, or any official rights holders. No copyrighted assets are used.

## Features

- **Games**: Dino Runner & Reaction Game
- **Music Player**: Lofi beats to vibe to (skip tracks with arrow buttons)
- **Interactive Screens**: Toggle between face, games, video player, and special images
- **YouTube Integration**: Watch videos on BMO's screen

## How to Use

### Run Locally

1. Download or clone this repo
2. Open `index.html` in your browser
3. That's it! No build process needed.

**OR** use a local server:
```bash
python -m http.server 5173
```
Then open http://localhost:5173

### Controls

- **Red Button (PLAY GAMES)**: Start games
- **Green Button (BACK)**: Return to face
- **Blue Dot**: Toggle NO SIGNAL screen
- **Video Button**: Play YouTube video
- **Music Controls**: Previous / Play-Pause / Next
- **D-pad**: Navigate menus & control games
- **Blue Rectangle Buttons**: Toggle File View & Blueprint images
